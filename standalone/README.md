# ViralCaption AI - Standalone Version (FREE)

A modern, standalone web application for generating viral captions with a sleek design and unlimited free access.

## Features
- **Unlimited Generations**: Generate as many captions as you want for free
- Generate 8 viral captions, 13 hashtags, and 1 hook sentence
- Support for TikTok, Instagram, YouTube, Twitter, Facebook, LinkedIn
- Copy to clipboard functionality
- Save generations to local storage
- Trending topics suggestions
- Modern dark UI with gradients and glassmorphism effects
- Mobile responsive
- User registration and login system
- Admin panel with full tracking capabilities

## Admin Features
- **Complete Oversight**: Track all user activities and financial data
- User management (view, edit, delete users)
- Detailed reports and analytics
- System settings configuration
- Real-time activity monitoring
- Revenue and subscription tracking

## How to Use
1. Open `index.html` in a web browser
2. Enter your video topic
3. Select platform
4. Click "Generate Captions"
5. First result is free, others are blurred
6. Click "Unlock all for €2.99" to go to checkout
7. Choose payment method and complete purchase
