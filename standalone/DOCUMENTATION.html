🔌 API Reference

Caption Generation Endpoint

POST /api/captions/generate Request Body: { "topic": "string", "platform": "tiktok|instagram|youtube|twitter|facebook|linkedin", "tone": "funny|inspirational|professional|casual|trendy|emotional", "keywords": "string (space-separated)" } Response: { "captions": ["caption1", "caption2", ...], "hashtags": ["#tag1", "#tag2", ...], "platformIdeas": ["idea1", "idea2", ...], "engagementHooks": ["hook1", "hook2", ...] }

Authentication Endpoints

POST /api/auth/register POST /api/auth/login POST /api/auth/logout GET /api/auth/verify POST /api/user/profile PUT /api/user/profile DELETE /api/user/account
ℹ️ Note: All endpoints require proper authentication headers for protected routes. Include JWT token in the Authorization header.

Error Handling

Standard Error Response: { "success": false, "error": "Error message", "code": "ERROR_CODE", "status": 400 }

⚡ Performance Optimization

Caching Strategy

  • Local storage for user captions (500 max items)
  • Session storage for temporary data
  • IndexedDB for large datasets (future implementation)
  • API response caching with 1-hour TTL

Optimization Techniques

  • Lazy loading for images
  • Debounced search input
  • Throttled scroll events
  • Memoized expensive calculations
  • CSS animations using GPU acceleration
  • Minimized DOM manipulation
  • Event delegation for dynamic elements

Performance Monitoring

// Check performance metrics window.addEventListener('load', () => { const perfData = performance.getEntriesByType('navigation')[0]; console.log('TTFB:', perfData.responseStart - perfData.requestStart); console.log('DOMContentLoaded:', perfData.domContentLoadedEventEnd); console.log('Page Load:', perfData.loadEventEnd); }); // Monitor via VCAi VCAi.performance.init();
✅ Best Practices: Use VCAi.utils.performance utilities to optimize your custom code. Always debounce user input and throttle scroll events.
